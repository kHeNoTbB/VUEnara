# 기본설정
## Node.js

노드 버전확인

``node --version``



실행

``node``



## NPM(Node Package Manager)

새로운 프로젝트/패키지 생성

``npm init``



패키지 설치

``npm install 패키지명`` 혹은 ``npm install``

* package.json 기반으로 패키지를 설치할 경우 npm install만 실행한다.



패키지 삭제

``npm uninsatll``



패키지 업데이트

``npm update``



스크립트 실행

``npm run 명령어``



## Vue-CLI

설치

``npm insatll -g @vue/cli``



실행

``vue create 프로젝트명``

``cd 프로젝트명``

``npm run serve``



## Webpack 설정파일 생성

package.json 파일 생성

``npm init``



webpack 설치

``npm install --save-dev webpack webpack-cli``



webpack 설정파일 생성

``touch webpack.config.js``
